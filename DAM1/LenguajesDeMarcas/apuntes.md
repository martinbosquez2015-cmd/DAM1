# Apuntes y cheatseets de Lenguajes de Programación


## O. Cosas Básicas:

### 0.1. Títulos, párrafos, listas, cosillas con el color
- Títulos: se maneja mediante encabezados, tienen que estar en orden
```
<html>
<h1>Encabezado de nivel 1</h1>
<h2>Encabezado de nivel 2</h2>
<h3>Encabezado de nivel 3</h3>
<h4>Encabezado de nivel 4</h4>
<h5>Encabezado de nivel 5</h5>
<h6>Encabezado de nivel 1</h6>
</html>
``` 
- Párrafos y saltos de linea
```
Se puede escribir sin poner nada
<p>Hay caracteres que tienen un código como este "&#x1559;(`&#x25bf;&#xb4;)&#x1557;"</p>
<br>Esto es para hacer un salto de linea
```
- Listas
1. Listas ordenadas:
```
<ol>
        <li><i>Miéntele</i> - bunkers.</li>
        <li><i>Motora</i> - Máquina camaleón.</li>
        <li><i>Tenías que ser tú</i> - Silvana Estrada</li>
</ol>
```
2. Listas no ordenadas:
```
<ul>
        <li type="square">Donnie Darko</li>
        <li >Nacho libre</li>
        <li>Spiderman (2002)</li>
    </ul>
```
3. Listas de definición:
```
<dl>
        <dt>Homer:</dt>
        <dd><b>1. </b>American bonehead.</dd>
        <dd><b>2. Pull a homer:</b> to suceeed despite idiocy.</dd>
        <dt>Homogeneus</dt>
        <dd><b>1. </b>Like in nature.</dd>
    </dl>
```
